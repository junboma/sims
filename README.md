## 乘务教育管理系统（SIMS）
## 简介

### 本项目采用前后端分离的方案：

#### 前端1(web)：Vue3.0+js+Element-Plus+Echars 搭建客户端

#### 前端2(wxapp)：微信原生小程序

#### 后端(Backend)：Django REST framework+Mysql 构建API接口
